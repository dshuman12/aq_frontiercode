# First Public Prototype
Core factory progression, draggable windows, and onboarding flow changes in the first public build.

## Summary
- Core factory loop is now playable end-to-end with mining, belts, processing, crafting, and research.
- Draggable utility windows are now part of the core UI.
- Auth/session flow now starts from a dedicated landing page.

## Details
### Highlights
- Early game progression now carries naturally from extraction to crafting.
- Window controls are cleaner and easier to reposition during play.
- New player entry points are clearer with explicit login/register routing.
