# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [1.0.0] - 2026-04

### Added
- Stable transcoding pipeline with HLS rendition ladder.
- Shared library access controls.
- Full search across courses, episodes, notes, and transcripts.

### Changed
- FSRS parameters tuned against real review data.

## [0.3.0] - 2025-09

### Added
- Notes panel with rich-text editor.
- Flashcard review surface backed by FSRS.
- Library shares (per-library invite tokens).

## [0.2.0] - 2024-11

### Added
- Course browsing UI, episode list, and Vidstack player.
- Bookmarks and progress tracking.
- Settings pages (account, sessions, libraries, connection).

## [0.1.0] - 2023-07

### Added
- First end-to-end vertical slice: scan a folder, list courses, play episodes.
- Auth via better-auth with email/password.
- Drizzle schema and migrations 0000-0001.
