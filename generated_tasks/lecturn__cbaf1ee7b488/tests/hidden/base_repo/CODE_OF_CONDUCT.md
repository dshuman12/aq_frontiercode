# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

Be kind. Assume good intent. Critique code, not people. Anyone who makes the
project unpleasant to participate in — through harassment, slurs, or repeated
bad-faith behavior — will be removed.

## Reporting

Email the maintainer privately at evilden982@gmail.com. Reports will be
handled confidentially and acted on within 7 days.
