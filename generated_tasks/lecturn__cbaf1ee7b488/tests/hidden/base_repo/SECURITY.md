# Security policy

## Supported versions

Only the latest minor release line receives security fixes.

| Version | Supported |
| ------- | --------- |
| 1.x     | yes       |
| < 1.0   | no        |

## Reporting a vulnerability

Please **do not** open a public issue. Instead, file a private advisory via
GitHub Security Advisories or email the maintainer at evilden982@gmail.com.

You should receive an acknowledgement within 72 hours. We aim to ship a fix
within 14 days for high-severity issues, longer for low-severity.
