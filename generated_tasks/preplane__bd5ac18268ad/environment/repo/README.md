# PrepLane

A web app for SAT practice and review.

## Features

- **Practice Rush** — endless practice questions with a timer, multiple choice and grid-in answers, back/forth navigation, Desmos calculator, and a reference sheet
- **Question Bank** — browse and filter questions
- **Tracker** — track answered questions and review history
- **Vocabulary** — wordbank, flashcards, and practice modes (including AI-assisted definition and usage feedback)
- **Bookmarks** — save questions for later
- **Lookup** — find a question by its ID

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## Stack

Next.js 15, React 19, TypeScript, Tailwind, shadcn/ui.

## Environment

Set the following in `.env.local` as needed:

- `NEXT_PUBLIC_BASE_URL` — public site URL
- `GA_KEY` — Google Analytics measurement ID (optional)
